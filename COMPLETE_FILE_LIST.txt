QUICK START
=================================================================

1. Create database:
   mysql -u root -p campus_event_hub < database/schema.sql

2. Configure database:
   Edit config/database.php

3. Start server:
   cd public && php -S localhost:8000

4. Open browser:
   http://localhost:8000

5. Register and enjoy!
